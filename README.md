# 🚀 NeuroReset - منصة التعافي من الإدمان  

## 📝 نظرة عامة  
NeuroReset هو تطبيق ويب يستخدم الذكاء الاصطناعي لإنشاء **مسار تعافٍ مخصص** لكل مستخدم، لمساعدته في الإقلاع عن العادات السيئة مثل التدخين والمخدرات، من خلال مهام يومية، مجتمع داعم، وجلسات مع مختصين.  

---

## 🔗 زيارة موقعنا  
📌 يمكنك تجربة NeuroReset الآن عبر الرابط التالي:  
[neuroreset.netlify.app](https://neuroreset.netlify.app/)

## ✨ الميزات الأساسية  
- ✅ **مسار تعافٍ مخصص بالذكاء الاصطناعي**  
- ✅ **مجتمع داعم للمستخدمين**  
- ✅ **إمكانية التواصل مع مختصين**  
- ✅ **مهام يومية ورسائل تحفيزية**  
- ✅ **إحصائيات لمتابعة التقدم**  

---

## 🛠 التقنيات المستخدمة  
- **Frontend:** Next.js, TypeScript, Tailwind CSS, Radix UI  
- **Backend & Database:** Supabase  
- **Authentication:** Supabase Auth  
- **State Management:** Zustand / React Context
